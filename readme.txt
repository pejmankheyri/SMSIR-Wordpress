== Installation ==
1. پوشه `SMSIR-Wordpress` را در مسیر `/wp-content/plugins/` کپی کنید
2. افزونه را از طریق منوی افزونه ها در وردپرس فعال کنید
3. تنظیمات افزونه را در بخش پیامک وردپرس sms.ir انجام دهید
